Effective prompts for Nano Banana image generation, organized by use case.

## Logos and Icons

```
"a minimalist logo of [subject], flat design, [color] on [background], vector style, clean lines"
```

## Photorealistic Images

```
"[subject], photorealistic, [lighting description], [camera angle], shot on [camera model], [atmosphere]"
```

## Illustrations

```
"[subject], digital illustration, [art style] style, [color palette], [mood]"
```

## UI Assets

```
"[element type] for a [app type] app, [design system] design, [color scheme], clean and modern"
```

## Transparent Assets

When using the `-t` flag, keep these in mind:
- Use subjects with clear, defined edges
- Avoid green elements in the subject
- Include "isolated on plain background" for cleaner results
- Simple subjects produce better cutouts than complex scenes

## Style Modifiers

- Lighting: "golden hour", "soft diffused light", "dramatic chiaroscuro", "neon-lit"
- Mood: "serene", "energetic", "mysterious", "playful"
- Texture: "smooth", "textured", "glossy", "matte"
